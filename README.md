## Build Process

Install all dependencies:

```
cd real-time-music-map
npm run install:all
```

Setup environment variables

```
change .env_sample to .env and setup your API/private keys
```

Run server:

```
cd /backend
npm run dev
```

Run react native app

```
cd /frontend
npx expo start
```
